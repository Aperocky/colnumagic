# Colnumagic Statistical Processing Toolbox

------

This tool box is strongly geared toward automation for very large, incomplete data in bad form. Starting from ingestion processing to automated feature selection. This package follow closely to the need of industry data applications. Mainly focusing on automating pre-processing and cleaning job for data science professionals.

## Pre Processing

This package include functions that reduces input sources API (REST/XML) into tree, which can then be further flattened into object by feature selection.

## Feature Selection

This package applies automatic correlation methods across all columns to pick out columns of interest. Useful when the data contains large number (200+) of columns and need arise to quickly develop this into a model.

## Description

All function are self developed, not used in production, but included for actual application in the industry. No proprietory code is contained.
